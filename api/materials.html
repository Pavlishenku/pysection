Materials API

Materials module for opensection

This module provides constitutive laws for structural materials according to Eurocodes (EC2 for concrete, EC3 for structural steel).

  • gamma_s – Coefficient partiel de sécurité

  • Es – Module d’Young (MPa)

  • include_hardening – Inclure l’écrouissage

  • k – Coefficient d’écrouissage (pente = k*Es)

stress(epsilon: float) → float[source]

Calcule la contrainte

s = Es·e pour |e| ≤ e_yk s = sign(e)·fyd pour e_yk < |e| ≤ e_ud (sans écrouissage) s = sign(e)·[fyd + Es^h·(|e|-e_yk)] pour e_yk < |e| ≤ e_ud (avec écrouissage)
